# Project Vision & Strategy - Portfolio Website

## 1. Mục tiêu chiến lược (Strategic Goals)
- **Thương hiệu cá nhân (Personal Branding):** Xây dựng một Portfolio trực tuyến cao cấp, chuyên nghiệp nhằm thu hút các Nhà tuyển dụng (Recruiters), Tech Leads và Khách hàng (Clients) trong lĩnh vực Dữ liệu và Tự động hóa.
- **Minh chứng năng lực (Competence Demonstration):** Thay vì chỉ viết sơ yếu lý lịch dạng chữ, trang web đóng vai trò như một phòng trưng bày sản phẩm sống động (Live Showcase) minh họa các dự án thực tế đã hoàn thành.
- **Đột phá kỹ thuật (Technical Quality):** Áp dụng quy chuẩn phát triển phần mềm chuẩn mực: Sử dụng TypeScript nghiêm ngặt, cấu trúc SCSS tối ưu, kiểm tra code tự động (ESLint/Prettier) và lập kế hoạch rõ ràng (Docs, Roadmap).

---

## 2. Điểm độc đáo / Khác biệt (Unique Selling Proposition - USP)

Trang web tập trung làm nổi bật sự giao thoa đặc biệt giữa 3 mảng năng lực chính của chủ sở hữu:

```
                  ┌───────────────────────────────┐
                  │        DATA ANALYTICS         │
                  │   (Trực quan hóa, Power BI)   │
                  └──────────────┬────────────────┘
                                 │
                 ┌───────────────┴───────────────┐
                 │          AUTOMATION           │
                 │  (Google Apps Script, Sheets) │
                 └───────────────┬───────────────┘
                                 │
                  ┌──────────────┴────────────────┐
                  │        WEB DEVELOPMENT        │
                  │     (React, TS, Vite, Sass)   │
                  └───────────────────────────────┘
```

- **Mảng Data Analytics & Engineering:** Khả năng biến đổi dữ liệu thô thành các chỉ số (KPIs) kinh doanh có giá trị, thiết kế dashboard tương tác (Power BI, Excel, Recharts).
- **Mảng Automation:** Tư duy tối ưu hóa quy trình, thiết lập hệ thống báo cáo tự động bằng Google Apps Script, Google Sheets nhằm tiết kiệm thời gian làm việc thủ công trong môi trường doanh nghiệp/sản xuất (Manufacturing).
- **Mảng Web Development:** Khả năng xây dựng các công cụ nội bộ (Internal Tools) hoặc trang web tối giản, hiệu năng cao và có trải nghiệm mượt mà.

---

## 3. Đối tượng mục tiêu (Target Audience)
- **Tech Leads / Hiring Managers:** Những người đánh giá cao tư duy tổ chức mã nguồn sạch (Clean Code), quy trình Git tiêu chuẩn và khả năng lập trình bằng React/TypeScript/Sass.
- **Business Owners / Operations Managers:** Những người quan tâm đến các giải pháp thực tế để tự động hóa công việc văn phòng, tiết kiệm thời gian và nâng cao năng suất của phòng ban.
